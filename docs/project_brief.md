# RogueLens project brief

## Project title

**RogueLens: A Python Microlensing Simulator for Rogue Planets**

## One-sentence description

RogueLens is an educational Python project that simulates how a free-floating planet could be detected through the temporary brightening of a background star caused by gravitational microlensing.

## Motivation

I wanted to build a project that connects astrophysics, programming, and mathematics. Rogue planets are especially interesting because they are difficult to observe directly. Microlensing gives a way to detect objects that may be dark, isolated, or too faint to see through ordinary imaging.

## Core question

How does the motion of a rogue planet in front of a background star create a measurable light curve?

## Methods

The project uses a simplified point-lens microlensing model. It calculates the angular separation between the lens and source over time, then converts that separation into magnification and observed flux.

## Outputs

- A Python package named `roguelens`.
- Example light curves.
- Comparison plots for different lens masses.
- A synthetic noisy observation.
- Documentation explaining the physics and limitations.

## Why it is useful for a student portfolio

This project shows independent learning, mathematical modeling, Python programming, scientific visualization, and the ability to explain a real astrophysical detection method in a clear way.
