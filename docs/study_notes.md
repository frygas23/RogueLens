# Study notes for understanding RogueLens

## Vocabulary

- **Rogue planet / free-floating planet**: a planetary-mass object not clearly bound to a host star.
- **Lens**: the object whose gravity bends light.
- **Source**: the background star whose light is magnified.
- **Microlensing**: gravitational lensing where the images are not resolved, but the source brightness changes over time.
- **Einstein radius**: the natural angular/physical scale of a lensing event.
- **Impact parameter (`u0`)**: closest lens-source separation in Einstein-radius units.
- **Einstein time (`tE`)**: approximate time for the lens-source separation to change by one Einstein radius.

## Intuition

The event is bright only when the lens and the background star are nearly aligned. The closer the alignment, the larger the magnification. The more massive the lens, the larger the Einstein radius, so the event usually lasts longer if the transverse velocity and distances are similar.
